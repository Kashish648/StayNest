# StayNest


STAYNEST is a full-stack Airbnb-style web application designed to simulate a real-world property rental marketplace. Users can explore listings, view property details, and manage accommodations through a responsive and interactive interface.

The project integrates Node.js, Express, MongoDB, React, and EJS to showcase modern web development practices, including RESTful routing, server-side rendering, and database-driven content.

This project focuses on building scalable backend systems and seamless frontend experiences, strengthening full-stack engineering skills through a practical, real-world application.

#Key Features

1.Dynamic property listings

2.CRUD operations for properties

3.MongoDB database integration

4.RESTful API architecture

5.Server-side rendering with EJS

6.React-based interactive UI

7.Responsive design
